# Quick Setup Guide

## 1. Hardware Connection
1. Connect sensors as per circuit_diagram.png
2. Power with 12V adapter
3. Connect Arduino via USB

## 2. Software Setup
1. Install Arduino IDE
2. Install required Libraries
3. Open homeguard.ino
4. Select Board: Arduino Nano
5. Upload code
   

## 3. Testing
1. Check LCD shows "(SENSOR)ON /DETECTED "
2. Test each sensor individually
3. Verify relay operation
